HTML

HTML (HyperText Markup Language) is the standard markup language used to create and structure web pages on the internet. It defines the layout and organization of content such as headings, paragraphs, images, links, tables, and forms using predefined tags.

CSS

CSS (Cascading Style Sheets) is a style sheet language used to control the appearance and design of web pages. It is used to style HTML elements by defining colors, fonts, spacing, layouts, and responsiveness, making websites visually attractive and user-friendly.

Java Script

JavaScript is a programming language used to make web pages interactive and dynamic. It allows developers to add functionality such as form validation, animations, sliders, pop-ups, and real-time content updates without reloading the page.
